## About Me

I’m a game developer from Indonesia, currently focused on building gameplay systems, horror game mechanics, UI/UX implementation, and 3D assets for Unreal Engine projects.

For the past 1.5 years, I’ve been developing and learning game production using **Unreal Engine 5**, mainly with **Blueprints** and **C++**. My goal is to create games that are visually appealing, optimized, creative, and enjoyable for many players.

I’m also the Co-Founder of **Monokotil Studio**, an indie game team focused on psychological and first-person horror games.

---

## What I Build

- Gameplay systems for Unreal Engine 5.
- Horror atmosphere tools and cinematic game features.

- Benchmarking and performance tools for Unreal Engine projects.
- Web portfolio and studio-related systems.

## Featured Projects

### DontIn — Current Horror Game Project

An original first-person psychological horror game project developed with **Unreal Engine 5** under **Monokotil Studio**, featuring modular horror triggers, interactive gameplay systems, dialogue flow, OS-style in-game interface, loading system, credits system, door portal mechanics, screenshot capture, and built-in benchmark tools.

### Monocots Horror Atmosphere Plugin

Custom Unreal Engine 5 horror atmosphere plugin featuring dynamic rain, fog, thunder, wetness, puddles, Niagara FX, audio atmosphere, and editor-friendly controls.

### Monokotil Studio Website

Portfolio and studio website built to showcase projects, team profiles, game development works, and studio branding.

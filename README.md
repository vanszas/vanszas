## About Me

I’m a game developer from Indonesia, currently focusing on Unreal Engine 5, gameplay systems, horror game mechanics, UI/UX implementation, and 3D asset creation.

For the past 1.5 years, I’ve been learning and building game projects with Unreal Engine using both Blueprint and C++. My goal is to create games that are visually appealing, optimized, creative, and enjoyable for many players.

I’m also the Co-Founder of **Monokotil Studio**, an indie game team focused on psychological and first-person horror experiences.
